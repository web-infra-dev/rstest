# maxConcurrency

- **类型：** `number`
- **默认值：** `5`
- **CLI：** `--maxConcurrency=10`

使用 `test.concurrent` 或 `describe.concurrent` 时允许同时运行的最大测试用例数量。

超过此限制数量的测试将在前序测试完成后排队运行。
