# browser

- **类型：**

```ts
type BrowserModeConfig = {
  enabled?: boolean;
  provider?: 'playwright';
  browser?: 'chromium' | 'firefox' | 'webkit';
  headless?: boolean;
  port?: number;
};
```

- **默认值：**

```ts
const defaultBrowser = {
  enabled: false,
  provider: 'playwright',
  browser: 'chromium',
  headless: true, // CI 环境；本地开发为 false
  port: undefined, // 随机可用端口
};
```

浏览器模式配置。启用后，测试将在真实浏览器中运行，而非 Node.js 环境。

## 常用配置（Recipes）

### 本地调试（推荐）

在本地开发时，通常希望打开浏览器窗口方便观察与调试：

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  browser: {
    enabled: true,
    headless: false,
  },
});
```

### CI 运行（推荐）

在 CI 环境中，通常以 headless 模式运行，并确保在执行测试前安装 Playwright 浏览器：

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  browser: {
    enabled: true,
    headless: true,
  },
});
```

```bash
npx playwright install --with-deps
```

### 多浏览器

如果你的项目需要验证跨浏览器一致性，推荐结合 [projects](/config/test/projects) 同时在多个浏览器中运行。完整示例请参考本页下方的“多浏览器测试”。

## 选项

### enabled

- **类型：** `boolean`
- **默认值：** `false`

启用浏览器模式。

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  browser: {
    enabled: true,
  },
});
```

:::tip
启用浏览器模式需要安装 `@rstest/browser` 包和 Playwright 浏览器。

```bash
npm add @rstest/browser -D
npx playwright install chromium
```

:::

### provider

- **类型：** `'playwright'`
- **默认值：** `'playwright'`

浏览器驱动提供者。目前仅支持 [Playwright](https://playwright.dev/)。

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  browser: {
    enabled: true,
    provider: 'playwright',
  },
});
```

### browser

- **类型：** `'chromium' | 'firefox' | 'webkit'`
- **默认值：** `'chromium'`

用于测试的浏览器类型。

- `chromium` - Google Chrome、Microsoft Edge
- `firefox` - Mozilla Firefox
- `webkit` - Safari

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  browser: {
    enabled: true,
    browser: 'firefox',
  },
});
```

使用前需要安装对应的浏览器：

```bash
# 安装 Chromium
npx playwright install chromium

# 安装 Firefox
npx playwright install firefox

# 安装 WebKit
npx playwright install webkit

# 安装所有浏览器
npx playwright install
```

### headless

- **类型：** `boolean`
- **默认值：** CI 环境为 `true`，本地开发为 `false`

是否以无界面模式运行浏览器。

### port

- **类型：** `number`
- **默认值：** `undefined`（自动选择可用端口）

浏览器模式 Dev Server 的端口号。

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  browser: {
    enabled: true,
    port: 5173,
  },
});
```

如果指定的端口已被占用，Rstest 会报错。如果未指定，将自动选择一个可用端口。

## 多浏览器测试

通过 [projects](/config/test/projects) 配置可以同时在多个浏览器中运行测试：

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  projects: [
    {
      name: 'chromium',
      browser: {
        enabled: true,
        browser: 'chromium',
      },
    },
    {
      name: 'firefox',
      browser: {
        enabled: true,
        browser: 'firefox',
      },
    },
    {
      name: 'webkit',
      browser: {
        enabled: true,
        browser: 'webkit',
      },
    },
  ],
});
```

## 与 Node 测试混合

可以同时配置浏览器测试和 Node.js 测试：

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  projects: [
    {
      name: 'browser',
      include: ['src/**/*.browser.test.ts'],
      browser: {
        enabled: true,
      },
    },
    {
      name: 'node',
      include: ['src/**/*.node.test.ts'],
      testEnvironment: 'node',
    },
    {
      name: 'jsdom',
      include: ['src/**/*.test.ts'],
      exclude: ['src/**/*.browser.test.ts', 'src/**/*.node.test.ts'],
      testEnvironment: 'jsdom',
    },
  ],
});
```
