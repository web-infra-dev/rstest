# unstubGlobals

- **类型：** `boolean`
- **默认值：** `false`
- **CLI：** `--unstubGlobals`

每次测试时恢复之前使用 [rstest.stubGlobal](/api/rstest/utilities#rsteststubglobal) 更改的所有全局变量。
