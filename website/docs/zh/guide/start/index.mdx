# 介绍

Rstest 是一个基于 Rspack 的测试框架，它为 Rspack 生态提供了全面、一流的支持，能够轻松集成到现有的 Rspack 项目中。

Rstest 提供兼容 Jest 的 API，同时提供对 TypeScript、ESM 等的原生、开箱即用支持，确保现代高效的测试体验。

## 💡 当前状态

Rstest 目前正处于积极开发阶段，目前尚无已发布的 npm 包。

我们欢迎开发者克隆代码仓库并在[本地运行](https://github.com/web-infra-dev/rstest/blob/main/CONTRIBUTING.md)，以探索正在开发中的功能。你可以查看代码库中的一些可用[示例](https://github.com/web-infra-dev/rstest/tree/main/examples)以试用 Rstest。

## 🔥 路线图

我们计划在 2025 年 6 月发布第一个预览版本，并于 2025 年晚些时候发布第一个稳定版本。

更多详情请参阅 [Rstest Roadmap](https://github.com/web-infra-dev/rstest/issues/85)。

## 🦀 Rstack

Rstack 是一个围绕 Rspack 打造的 JavaScript 统一工具链，具有优秀的性能和一致的架构。

<img
  src="https://assets.rspack.dev/rstack/rstack-overview.png"
  alt="Rstack"
  width="820"
/>

Rstack 包含以下工具：

| 名称                                                  | 描述           |
| ----------------------------------------------------- | -------------- |
| [Rspack](https://github.com/web-infra-dev/rspack)     | 打包工具       |
| [Rsbuild](https://github.com/web-infra-dev/rsbuild)   | 构建工具       |
| [Rslib](https://github.com/web-infra-dev/rslib)       | 库开发工具     |
| [Rspress](https://github.com/web-infra-dev/rspress)   | 静态站点生成器 |
| [Rsdoctor](https://github.com/web-infra-dev/rsdoctor) | 构建分析工具   |
| [Rstest](https://github.com/web-infra-dev/rstest)     | 测试框架       |
