# passWithNoTests

- **Type:** `boolean`
- **Default:** `false`
- **CLI:** `--passWithNoTests`

Pass when no tests are found.

By default, Rstest marks test failures when no test suites are found. You can allow the test to pass when no tests are found by setting `passWithNoTests` to `true`.
