# unstubEnvs

- **Type:** `boolean`
- **Default:** `false`
- **CLI:** `--unstubEnvs`

Restores all `process.env` values that were changed with [rstest.stubEnv](/api/rstest/utilities#rsteststubenv) before every test.

import { Tab, Tabs } from '@theme';

<Tabs defaultValue='rstest.config.ts'>
  <Tab label="CLI">

  </Tab>
  <Tab label="rstest.config.ts">
```ts

  unstubEnvs: true,
});
```
