# hookTimeout

- **Type:** `number`
- **Default:** `10_000`
- **CLI:** `--hookTimeout=10000`

Default timeout of [hook](/api/test-api/hooks) in milliseconds. `0` will disable the timeout.

Rstest determines the hook timeout based on the following priority order:

1. [timeout](/api/test-api/hooks) configuration in each test case
2. Timeout configured in the test file via [`rstest.setConfig({ hookTimeout })`]
3. `--hookTimeout` option in CLI
4. `hookTimeout` option in the configuration file
5. Default value of `10_000` milliseconds
