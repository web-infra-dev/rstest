# resetMocks

- **Type:** `boolean`
- **Default:** `false`
- **CLI:** `--resetMocks`

Automatically reset mock state before every test.

When `resetMocks` enabled, rstest will call [.resetAllMocks()](/api/rstest/mockFunctions#rstestresetallmocks) before each test.
