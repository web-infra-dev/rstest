# restoreMocks

- **Type:** `boolean`
- **Default:** `false`
- **CLI:** `--restoreMocks`

Automatically reset mock state before every test.

When `restoreMocks` enabled, rstest will call [.restoreAllMocks()](/api/rstest/mockFunctions#rstestrestoreallmocks) before each test.
