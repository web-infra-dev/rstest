# unstubGlobals

- **Type:** `boolean`
- **Default:** `false`
- **CLI:** `--unstubGlobals`

Restores all global variables that were changed with [rstest.stubGlobal](/api/rstest/utilities#rsteststubglobal) before every test.

import { Tab, Tabs } from '@theme';

<Tabs defaultValue='rstest.config.ts'>
  <Tab label="CLI">
  ```bash

  ```
  </Tab>
  <Tab label="rstest.config.ts">
```ts
import { defineConfig } from '@rstest/core';

export default defineConfig({
  unstubGlobals: true,
});
```
