# printConsoleTrace

- **Type:** `boolean`
- **Default:** `false`
- **CLI:** `--printConsoleTrace`

Print console traces when calling any console method, which is helpful for debugging.

```ts title="rstest.config.ts"
import { defineConfig } from '@rstest/core';

export default defineConfig({
  printConsoleTrace: true,
});
```

:::note
When [disableConsoleIntercept](/config/test/disableConsoleIntercept) is set to `true`, `printConsoleTrace` will be ignored.
:::
